1. 树形大纲 + 侧栏详情（Nean Project / Alios）

左侧是无限嵌套的树形任务列表，右侧或下方是选中节点的详情面板。自动汇总子任务进度到父节点。

3. 思维导图式 WBS（Easy WBS / Miro / Creately）

以思维导图/气泡图形式展示 WBS 层级，节点可拖拽重排、缩放。适合演示/规划阶段。

4. 表格式 WBS（Asana / Monday.com / ClickUp）

可排序表格，缩进表达层级。支持多列（状态/优先级/截止日/负责人），适合数据密集型团队。
