数据模型扩展

interface WBSNode {
// 继承现有 TaskSubtaskNode
  id: string
  wbsCode: string // CY-01.01.01
  name: string
  nodeLevel: 'project' | 'workPackage' | 'task' | 'subtask'
  status: TaskStatus
  progress: number // 子节点自动汇总
  plannedStart: string | null
  plannedEnd: string | null
  duration: number // 天
  assignee: string | null
  parentId: string | null
  children: WBSNode[]
}

WBS 编码自动生成规则

CY-01 项目前缀 + 两位序号 工作包
CY-01.01 父编码 + 两位序号 任务
CY-01.01.01 递进编号 子任务

进度汇总规则

  • 叶子节点（子任务）：手动填写进度 0-100%
  • 父节点（任务/工作包）：= 子节点进度均值（按必做子任务加权）
  • 项目节点：= 工作包进度均值
  • 有未完成的必做子任务时，父节点状态不得为"已完成"
  • 关键路径上的任务显示特殊标记
