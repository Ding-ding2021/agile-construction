# 设计阶段

## 目标

将需求转化为可视化的设计方案，确定产品的视觉风格和交互方式。

## 关键活动

### 1. 交互设计

- **信息架构**：梳理页面层级与导航结构
- **流程设计**：绘制用户任务流程图
- **线框图**：产出低保真页面布局
- **交互说明**：标注状态变化与反馈

### 2. 视觉设计

- **设计规范**：定义色彩、字体、间距系统
- **高保真稿**：产出完整视觉设计稿
- **组件库**：提取可复用设计组件
- **切图标注**：输出开发所需资源

### 3. 设计评审

- **内部评审**：设计团队内部把关
- **交叉评审**：开发、产品参与评审
- **用户测试**：快速可用性测试

## 设计规范模板

```
## 色彩系统

| 类型 | 色值 | 用途 |
|-----|------|-----|
| 主色 | #1890FF | 主要按钮、链接 |
| 成功 | #52C41A | 成功状态、通过 |
| 警告 | #FAAD14 | 警告提示 |
| 错误 | #F5222D | 错误状态、删除 |
| 中性色 | #262626-#F5F5F5 | 文本、背景、边框 |

## 字体规范

| 层级 | 字号 | 字重 | 用途 |
|-----|------|-----|-----|
| 标题 H1 | 24px | Bold | 页面主标题 |
| 标题 H2 | 18px | Medium | 区块标题 |
| 正文 | 14px | Regular | 常规文本 |
| 辅助 | 12px | Regular | 说明文字 |

## 间距系统

基础单位：8px

| Token | 值 | 用途 |
|-------|---|-----|
| xs | 4px | 紧凑间距 |
| sm | 8px | 默认间距 |
| md | 16px | 区块间距 |
| lg | 24px | 大模块间距 |
| xl | 32px | 页面边距 |
```
