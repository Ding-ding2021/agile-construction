# Harness 工程框架 — 总纲

> 一个可复用的 opencode 驱动开发工程框架，标准化 AI 和人类协作的每一项要素。

## 一句话

**七阶段流水线串联四专业角色，三组评审把关，72 技能武装，71+ 文档支撑，51 指标量化，人类最终决策。**

## 三大原则

| 原则             | 说明                                            |
| ---------------- | ----------------------------------------------- |
| **线性不分支**   | 所有任务走同一条七阶段管道，只有执行深度的区别  |
| **人类最终决策** | 三组评审给出结论，但批准权永远在人类手中        |
| **自进化闭环**   | 阶段 7 把交付经验反哺回阶段 1，下次任务自动受益 |
