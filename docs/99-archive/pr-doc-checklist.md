# 文档治理 PR 检查清单

## Clause 1. 合并前检查项

### 1.1 [强制] 文档范围

**1.1.1 [强制]** 仅在 `docs/` 目录新增或修改业务文档。

**1.1.2 [强制]** 同主题无多份 active 冲突文档。

**1.1.3 [强制]** 历史文档已迁入 `docs/99-archive/` 或标记 `deprecated`。

**1.1.4 [推荐]** 若影响设计规范，已同步 `design-specification` 与 `design-checklist`。

### 1.2 [强制] 文档质量

**1.2.1 [强制]** active 文档 Frontmatter 字段完整。

**1.2.2 [强制]** `docs/README.md` 已同步登记新增/变更文档。

**1.2.3 [强制]** 文档链接均为有效 `docs/...` 路径。
