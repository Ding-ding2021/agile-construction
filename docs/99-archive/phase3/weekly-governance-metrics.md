## 阶段3周指标与评审口径（调度 / 资源 / 结算）

### 1. 指标目标

- **调度域**：任务派单效率、拒单回退率、待提交阻断清零速度
- **资源域**：可分配资源占比、高负载人员占比、关键资质覆盖率
- **结算域**：草案差异处理时效、待复核项数量、结算确认率

### 2. 指标定义

| 领域 | 指标           | 计算口径                                      | 周目标     |
| ---- | -------------- | --------------------------------------------- | ---------- |
| 调度 | 派单响应时长   | 从“待分配”到“待执行/执行中”的平均时长         | <= 24h     |
| 调度 | 拒单回退率     | 拒单回退次数 / 派单总次数                     | <= 10%     |
| 调度 | 待提交阻断率   | 待提交且资料不齐任务数 / 在制任务数           | <= 15%     |
| 资源 | 可分配占比     | availability=assignable 人数 / 总人数         | >= 60%     |
| 资源 | 高负载占比     | busy 人数 / 总人数                            | <= 25%     |
| 资源 | 关键资质覆盖率 | 覆盖关键工种人员数 / 关键工种需求数           | >= 95%     |
| 结算 | 差异项处理时长 | 差异项创建到“处理中/关闭”的平均时长           | <= 48h     |
| 结算 | 待复核项数量   | 状态=需复核 的差异项总数                      | 周环比下降 |
| 结算 | 结算确认率     | settlementStatus=已确认 项目数 / 可结算项目数 | >= 80%     |

### 3. 评审输出格式（每周固定）

- **本周达成**：达标指标 / 未达标指标
- **问题诊断**：前三大阻断及根因
- **动作清单**：下周3个优先动作（带完成标准）
- **风险追踪**：存量风险、增量风险、已关闭风险

### 4. 数据采集建议

- 调度：`TaskManagementPage` + 任务流转日志
- 资源：`ResourcePoolPage` + 人员可分配状态
- 结算：`ContractSettlementPage` + 差异项与复核状态
- 审计：`/audit/logs` 作为关键动作佐证

### 5. DoD（周评审通过）

- 三域指标均有数据来源和计算说明
- 未达标项均有责任动作与截止时间
- 下周入口可直接执行，不依赖口头解释
