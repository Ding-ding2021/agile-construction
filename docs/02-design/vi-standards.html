中性色使用规则

底色

✅ 正确
bg-background — 页面背景 bg-card — 卡片背景 bg-muted — 次要表面
❌ 禁止
bg-white — 用 bg-background bg-black — 用 bg-background (.dark) bg-neutral-50 — 那是 background

文字

✅ 正确
text-foreground — 主文字 text-muted-foreground — 辅助文字
❌ 禁止
text-black / #000 — 用 text-foreground text-white / #FFF — 用 text-foreground (.dark) text-neutral-900 — 用 text-foreground

灰色过渡

✅ 用色阶分层
bg-neutral-50 → 100 → 200
❌ opacity 模拟灰色
bg-black/5 bg-foreground/10
03

语义色 · 使用总则

颜色只用于表达语义状态，不用作装饰。始终使用 CSS 变量语义别名，不直接使用色阶类名。

foreground
主文字
muted-foreground
辅助文字
success
成功 / 已完成
destructive
删除 / 错误
accent (blue)
强调 / 链接

正确 vs 禁止

✅ 正确
<Button variant="destructive">删除</Button> <Badge variant="outline">进行中</Badge> <div className="text-destructive">错误提示</div>
❌ 禁止
<Button className="bg-red-500">删除</Button> <span className="text-orange-600">进行中</span> <div style={ color: '#ef4444' }>错误</div>
04

色板系统

项目支持 5 套可选色板 + 1 套默认 neutral 色板，通过 html 类名切换。色板只影响 primary、chart、ring、sidebar 等强调色，中性色不受影响。

theme-blue

theme-green
暖绿 primary · 项目/环境主题
theme-orange
暖橙 primary · 紧急/高危
theme-purple
紫 primary · 特权/Admin
theme-stone
暖灰 primary · 阅读/文档
neutral（默认）
纯黑白 · 无额外类名
05

暗色模式

颜色通过 CSS 变量自动适配 .dark 类。所有颜色使用语义变量（bg-background），暗色自动反转。

✅ 规则
默认亮色，暗色由用户偏好决定 使用语义变量，暗色自动适配 色板 × 暗色可正交组合
❌ 例外
仅亮暗使用不同色阶时才需 dark: 前缀 不要手动为每个元素写 dark:xxx
