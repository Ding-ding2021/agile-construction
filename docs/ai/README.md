# AI 合约索引

> 本目录为 AI Agent 提供结构化模块合约。每个合约 ≤ 200 行，纯表格/清单格式。
> 合约由 `document-sync` 技能自动从人类文档提取，双向链接到 `docs/` 下的原文。

## 子目录

| 目录         | 用途                                 | 读者        |
| ------------ | ------------------------------------ | ----------- |
| `contracts/` | 模块合约（每模块一个文件）           | AI 按需加载 |
| `knowledge/` | 从交付中提取的可复用模式、规则、决策 | AI 启动加载 |
| `context/`   | 当前项目状态、活跃任务               | AI 启动加载 |
