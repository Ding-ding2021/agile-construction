    区域命名表

    CSS 类名区域职责对应组件
    .pm-app根容器全页 flex 容器，min-height: 100vhAppShell
    .pm-sidebar侧边栏256px 固定宽度导航AppSidebar
    .pm-header页头64px 页面标题 + 搜索 + 用户信息PageHeader
    .pm-workspace工作区右侧弹性区域，flex: 1AppShell
    .pm-body主体滚动容器，calc(100vh - header)，内边距 24px页面自行管理
    .pm-stats-row统计行统计卡片网格（7 列）StatsCards
    .pm-content内容区表格/卡片/视图的具体内容页面自行管理
