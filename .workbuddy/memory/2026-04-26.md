## 设施管理页面布局修复

### 问题

设施管理页面侧边栏与主窗口重叠，侧边栏压在主窗口上。

### 根因

`.fm-page` 使用了 CSS Grid 布局（`grid-template-columns`），但 `AppSidebar` 组件渲染的 `<aside class="pm-sidebar">` 有 `z-index: 100`，而 `.fm-main` 只有 `z-index: 1`，导致侧边栏在 z 轴上覆盖了主内容区。

### 修复

- `facility-management.css`: `.fm-page` 从 `display: grid` 改为 `display: flex`（匹配 `ProjectManagementPage` 的 `.pm-app` 模式）
- `.fm-main` 增加 `flex: 1` 填充剩余空间
- 移除 `@media (max-width: 768px)` 中的 `grid-template-columns` 规则
- lint: 0 errors ✅
